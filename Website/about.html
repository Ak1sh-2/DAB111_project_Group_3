About the Car Sales Dataset

This project uses a car sales dataset to manage and display details about cars.

Dataset Details:

Variable Description
Brand Car manufacturer
Price Sale price of the car
Body Body type (e.g., sedan, SUV)
Mileage Distance covered (in km or miles)
Engine Version Version of the car engine
Engine Type Type of engine (e.g., petrol, gas)
Registration Registration status
Year Manufacturing year
Model Car model
